# Automate-AWS-using-Terrraform
This DevOps project shows how to automate AWS cloud infrastructure using Terraform. It involves creating a VPC with subnets, internet gateway, and route tables, configuring security groups, setting up S3 storage, and launching Ubuntu EC2 instances to host a web application. An Application Load Balancer (ALB) is used to distribute traffic across the instances. Finally, `terraform destroy` is used to remove all resources to avoid extra AWS costs.
